# PDF 편집기 (전자책 수정용)

AI로 만든 전자책 PDF를 **단어/문장 단위**로 수정하는 웹 편집기입니다.  
한글 "안녕하세요"가 글자 단위가 아니라 **한 덩어리**로 선택·수정됩니다.

- **지원 용량**: 최대 100MB
- **기술**: React, TypeScript, PDF.js(렌더/텍스트 추출), pdf-lib(저장)

## 사용법

1. `npm install` 후 `npm run dev` 로 실행
2. **PDF 열기**로 파일 선택 (최대 100MB)
3. 페이지를 클릭해 이동
4. **본문에서 수정할 텍스트 덩어리**를 클릭
5. 오른쪽 패널에서 내용 수정 후 **적용**
6. **수정본 저장**으로 편집 반영된 PDF 다운로드

## 한글 저장 시 참고

저장 시 새 텍스트는 현재 **Helvetica**로 그려집니다.  
한글은 Helvetica에서 깨질 수 있으므로, 나중에 **한글 폰트(예: Noto Sans KR)를 임베드**하는 방식으로 확장할 수 있습니다.
