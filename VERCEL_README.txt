이 프로젝트는 Vite + React PDF 편집기입니다.
Vercel에 배포하려면 먼저 이 폴더 전체를 GitHub에 푸시해야 합니다.

푸시 방법 (한 번만 실행):
  cd C:\Users\user\pdf-editor
  git init
  git add .
  git commit -m "Initial commit"
  git remote add origin https://github.com/seunghwan-6829/pdf-editor.git
  git branch -M main
  git push -u origin main

푸시 후 Vercel에서 다시 "Deploy" 하면 됩니다.
